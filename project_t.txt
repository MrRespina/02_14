Spring or Spring Boot 프로젝트

	- 무엇을 주제로 선정해야 하는가?

	- 무슨 기술 스택을 사용해야 하는가?
	>>> 기본적으로 사용할 것 : Java Spring or Java SpringBoot, JSP, HTML, JavaScript, OracleDB, 
	>>> 세부적인 내용 : Spring 쪽은 기본적인 세팅으로 Maven,MyBatis 사용
	>>> JSP,HTML,JavaScript같은 Front-END 단은 ajax나 jquery를 사용하여 동적인 페이지로 만들어 사용.
	>>> OracleDB를 사용해야하나? - MySQL도 되겠지만 팀원들이 잘 못쓸 듯.
	>>> 이 외에도 주제가 정해진다면, 주제에 따라 추가로 사용해야 할 것들이 있을 것.
	>>> 아마 클라우드 플랫폼은 시간이 남으면 넣을 듯 하고, 네이버 SENS 인증시스템은 넣는 게 좋을 것 같음.
	>>> 

구글 클라우드 플랫폼 사용하여 300$(사용량 따라 차감 있음.)어치의 VMWare 사용가능.
이 기능을 사용하여 호스팅 서버를 만들 수 있음.
외부 호스팅 서버를 만들면, 굳이 집에서 서버를 켜놓지 않아도 외부에서 접속&테스트 가능할 것으로 예상.
이 외에도 이력서에 직접적인 URL 등을 사용함으로써 결과물을 직관적으로 보여줄 수 있을 것.

회원가입에 인증을 넣을 수 있음.
네이버 클라우드 API SENS 사용 < 한 달에 50건 무료
인증 시스템을 넣음으로써 실제로 회원가입하는 느낌을 낼 수 있고 배웠던 것에서 발전하여 사용 가능할 것.

Selenium을 사용하여 Spring에서 동적 페이지를 크롤링 가능.
